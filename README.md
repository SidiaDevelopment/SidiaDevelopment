# Hi, I'm Marvin

Game & web developer out of Berlin. Around 13 years in the industry, currently building mobile roguelite games at **Swift Games** as a Senior Unity Developer.

## What I do

- **Games** — Unity / C#, shipped titles include [Heroes vs Hordes](https://apps.apple.com/us/app/heroes-vs-hordes-survival-rpg/id1608898173), [The Walking Dead: Aftermath](https://apps.apple.com/pl/app/the-walking-dead-aftermath/id6751237805), [Pull the Pin](https://apps.apple.com/us/app/pull-the-pin/id1496150467) (250M+ downloads), [Parking Jam 3D](https://apps.apple.com/us/app/parking-jam-3d/id1498229533) (130M+), and a long tail of soft-launched prototypes.
- **Web** — TypeScript / React / Next.js. Worked on online shops and B2B applications at Popcore, ReactionLink and CHECK24.

## What I do for fun

- Admin of the official German **League of Legends** and **Valorant** Discord communities (~250k members combined). I also wrote the freelance moderation bot Riot Games uses on the LoL server.
- Built a Discord bot for [Maxroll's Path of Exile Trading server](https://discord.com/invite/path-of-exile-trading-530668348682403841) that syncs service offers and handles vouches.
- Built [acronym.sidia.net](https://acronym.sidia.net) and a Reddit bot ([u/PoE_Acronym_Bot](https://www.reddit.com/user/PoE_Acronym_Bot)) that auto-defines Path of Exile acronyms for newer players.
- Self-host a small infra (5 servers, Linux + Docker + Nginx + Systemd) running my websites, bots, automations, backups and monitoring.
- Run [share.sidia.net](https://share.sidia.net), a code-sharing site with ShareX integration.
